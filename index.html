EXPENSE TRACKER Website
Expense Tracker project that seamlessly integrates various technologies to offer robust functionality and user-friendly experience. Leveraging Node.js for its backend, Bootstrap for frontend design, and Express.js for efficient routing, this project exemplifies the fusion of cutting-edge technologies. Hosted on an AWS EC2 instance, the Expense Tracker ensures scalability and reliability. It offers a suite of features including CRUD operations for managing expenses effortlessly. Additionally, it provides a convenient expense download functionality, empowering users to seamlessly export their financial data for analysis and record-keeping. This project not only demonstrates my proficiency in utilizing modern tech stacks but also underscores my commitment to delivering practical solutions that meet diverse user needs.

CHAT APP Website
The chat application leveraging Node.js, Express.js, and Socket.IO, this application delivers real-time communication capabilities with seamless integration of websockets. The use of Sequelize ORM ensures efficient database management, facilitating smooth interactions between users, chat histories, and groups. Additionally, the incorporation of cron jobs enhances functionality with scheduled tasks. Hosted on AWS EC2, this application is scalable and reliable, capable of handling diverse user interactions. With a user-friendly interface crafted using Bootstrap and enhanced with CORS and cookie-parser middleware, the application ensures a secure and intuitive user experience. This project not only showcases my proficiency in backend development and database management but also highlights my ability to create engaging and responsive web applications tailored to modern communication needs.

ECOM WEBSITE Website
In my portfolio, I'm excited to present a dynamic e-commerce application built with Express.js and MongoDB. This project utilizes the power of Node.js to create a seamless backend infrastructure, while Mongoose provides a robust interface for interacting with MongoDB. Leveraging the Express framework, the application efficiently handles routing and middleware functionality. The user interface, crafted using the EJS templating engine and styled with custom CSS, offers an intuitive shopping experience. Additionally, the application features user authentication, allowing personalized interactions based on user profiles. Hosted on the cloud platform, this application ensures scalability and reliability. Through continuous integration of user feedback and iterative development, this project exemplifies my dedication to creating engaging and user-centric web applications.
